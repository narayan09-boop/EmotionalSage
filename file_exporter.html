Emotion-Aware Content Recommender - Source Files

Copy each file section below to recreate the application locally.

📁 Directory Structure
emotion-recommender-app/ ├── app.py ├── emotion_analyzer.py ├── movie_recommender.py ├── youtube_recommender.py ├── recommendation_engine.py ├── database.py ├── playlist_manager.py ├── .streamlit/ │ └── config.toml └── requirements.txt
📋 Installation Instructions
1. Install Python 3.11+ 2. Create project directory: mkdir emotion-recommender-app && cd emotion-recommender-app 3. Install dependencies: pip install streamlit pandas psycopg2-binary textblob vadersentiment requests 4. Set environment variables (see below) 5. Create all files from sections below 6. Run: streamlit run app.py --server.port 5000
🔑 Environment Variables
DATABASE_URL=postgresql://user:password@host:port/database YOUTUBE_API_KEY=your_youtube_api_key TMDB_API_KEY=your_tmdb_api_key (optional) SPOTIFY_CLIENT_ID=your_spotify_client_id (optional) SPOTIFY_CLIENT_SECRET=your_spotify_client_secret (optional)
